# Ejercicio 1 - Visualizador de Productos

## Qué contiene este proyecto
- `index.html` — página principal.
- `styles.css` — estilos básicos.
- `app.js` — lógica para cargar, filtrar y ordenar productos (usa fetch + async/await).
- `data/productos.json` — datos de partida (he añadido 3 productos extra).

## Contenido añadido al JSON
Se han añadido 3 productos con `id` p_extra_01, p_extra_02 y p_extra_03 para ampliar las categorías.

## Cómo desplegar en Netlify (pasos rápidos)
1. Crea un repositorio en GitHub con todo el contenido del proyecto (o usa tu repo existente).
2. En Netlify: "New site from Git" → conecta GitHub → selecciona el repo → deploy.
3. Opcionalmente, puedes arrastrar la carpeta **`/`** del proyecto a Netlify Drop: https://app.netlify.com/drop
4. Cuando Netlify termine el deploy, copia la URL que te da (algo como `https://<tu-sitio>.netlify.app`) y pégala aquí para que la incluya en el README.

> No puedo subir el proyecto a Netlify por ti desde este entorno. Por favor, despliega siguiendo los pasos anteriores y pega la URL resultante en el README en la sección **"URLs de Netlify"**.
